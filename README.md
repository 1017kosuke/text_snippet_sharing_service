# SharePet 🐾

A simple web application for sharing code snippets.

---

## 🇺🇸 English

### Overview
SharePet allows users to create and share code snippets via unique URLs.

### Features
- Create snippets
- Share via link
- Syntax highlighting (Monaco Editor)
- Language switch (EN / JP)

### Tech Stack
- PHP
- MySQL
- JavaScript
- Monaco Editor

---

## 🇯🇵 日本語

### 概要
SharePetはコードスニペットをURLで簡単に共有できるWebアプリです。

### 機能
- スニペット作成
- URL共有
- シンタックスハイライト（Monaco Editor）
- 言語切り替え（英語 / 日本語）

### 技術スタック
- PHP
- MySQL
- JavaScript
- Monaco Editor
